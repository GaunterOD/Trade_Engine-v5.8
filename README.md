# 🎩 Gaunter-O-Dimm Engine v6.0
**고빈도 유동성 추적 및 비선형 시장 에너지 융합 레이더 시스템 명세서**

## 1. 설계 철학 (Philosophy)
본 시스템은 자동 매매 봇이 아닌, 중장기적 시장 방향성을 선제적으로 포착하는 **고성능 심층 레이더(Deep-Radar)**이다. 최종 매매 판단은 인간의 직관과 차트 검증을 거치며, 본 엔진은 전 세계 최대 유동성 공급원인 Binance와 BitMEX의 실시간 피드를 기반으로 자본의 진성 유입(Authentic Capital Inflow)과 고래들의 의도를 수학적으로 해독하는 데 집중한다.

## 2. 고차원 수치 분석 모델 (Mathematical Framework)

### 2.1. 비선형 중력 벡터 (Non-linear Gravity Vector, GV)
다중 시계열(1D, 4H, 1H) 데이터의 지수적 가중치를 결합하되, 각 타임프레임별로 독립적인 **히스테리시스 버퍼(Hysteresis Threshold)**를 적용한다.
단순 가격 교차를 넘어서, 노이즈 구간(Whipsaw)을 필터링하고 진정한 추세 확립 시에만 방향성(±1)을 승인한다.
* GV = Σ [sgn(P_t - EMA_t) * Hysteresis_filter]

### 2.2. 시장 흡수 지수 (Market Absorption Index, MAI)
시장가 주문의 불균형(CVD)을 통계적 정규분포(Z-Score)로 치환하여 해석한다. 
극단적 수치(|Z| > 2.0) 도달 시, 이를 단순 변동성이 아닌 스마트 머니에 의한 의도적 매집/투하 구간으로 정의한다.
* MAI = (ΔVol_net - μ) / σ

### 2.3. 유동성 가속도 모멘텀 (Liquidity Acceleration Momentum, LAM)
시간의 흐름에 따른 MAI의 순간 변화율(미분값)을 추적한다. 
단기 윈도우 내에서 수급의 가속도가 특정 임계치를 돌파할 경우, 기습적인 자본 투입(폭발적 매수/매도세)으로 간주하고 즉각적인 경보를 발생시킨다.
* LAM = d(MAI) / dt

### 2.4. 복합 시장 에너지 강도 (Market Energy Power, MEP)
추세 점수(GV), 수급 지수(MAI), 그리고 미결제약정(OI)의 절대적/상대적 팽창률을 다변수 함수로 결합하여 롱(Bull)과 숏(Bear)의 실질적인 파워를 점수화한다.
방향성, 거래량, 파생상품 에너지가 3위 일체로 동기화될 때 가장 높은 스코어를 부여한다.
* MEP = α(GV) + β(MAI) + γ(OI_Δ)

## 3. 실전 운용 프로토콜 (Operation Protocol)

* **Step 1: 크로마-싱크 시각화 (Chroma-Sync HUD)**: 
  서버에서 수집된 방대한 데이터를 로컬 환경으로 전송, 양음(초록/빨강) 스펙트럼과 복합 지수를 통해 유동성 유입/유출을 직관적으로 파악한다. 시스템은 엄격한 시간 동기화(Time-Sync) 알고리즘에 의해 오차 없이 작동한다.
* **Step 2: 신호 무결성 및 연속성 검증 (Signal Integrity & Continuity)**: 
  단발성 휩쏘에 속지 않기 위해, 추세 방향성이 특정 N주기 이상 흔들림 없이 연속으로 유지되는지 교차 검증한다. 확정된 추세 진입 시에만 '진성 추세'로 알람을 발생시킨다.
* **Step 3: 스마트 쿨다운 및 트랩 감지 (Smart Cooldown & Trap Detection)**: 
  펀딩비(과열/포화)와 OI 이탈률을 계산하여 롱/숏 스퀴즈 트랩을 사전 경고하며, 무의미한 중복 경고를 막기 위한 비동기식 알람 쿨다운 시스템이 가동된다.

## 4. 보안 및 기밀 유지 (Strict Confidentiality)
* 본 엔진은 2원화(데이터 수집 서버 / 분석 HUD) 구조로 운영되며, 승인되지 않은 환경에서의 내부 코어 파라미터(Threshold 값 등) 및 데이터 로깅 유출을 엄격히 금지한다.
